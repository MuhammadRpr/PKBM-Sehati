## 🔧 Customization Examples

### 1. Update Site Name
Edit `lib/constants.ts`:
```typescript
SITE_CONFIG.name = "Nama Organisasi Anda"
```

### 2. Change Colors
Edit `tailwind.config.ts`:
```typescript
primary: {
  600: '#yourcolor'
}
```

### 3. Add Article
Create `content/blog/artikel-baru.mdx`:
```mdx
---
title: "Judul"
slug: "artikel-baru"
---
# Content here
```

### 4. Update Program
Edit `content/programs.json`:
```json
{
  "id": "paket-a",
  "title": "Paket A",
  ...
}
```
